Caricamento Massivo CSV

Come preparare il file CSV:

  1. Scarica il template CSV per vedere la struttura richiesta
  2. Compila il file con i dati delle auto (marca, modello, anno, prezzo, tipo, colore, km, carburante, descrizione, immagine)
  3. Carica il file completato
  4. Verifica l'anteprima e conferma l'importazione
