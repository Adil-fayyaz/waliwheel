Autenticazione con Google

Firebase Authentication (CDN v9) - Sito statico

Se il popup è bloccato, useremo il reindirizzamento automatico.

Guida rapida:

  1. Apri Firebase Console → Authentication → Metodo di accesso → abilita Google.
  2. Impostazioni Authentication → Domini autorizzati → aggiungi il dominio Vercel (es. mio-progetto.vercel.app).
  3. Apri il tuo progetto → Impostazioni (icona ingranaggio) → Configura app Web → copia l'firebaseConfig e incollalo in app.js dove indicato.
